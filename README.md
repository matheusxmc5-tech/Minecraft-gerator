# Minecraft Server Generator

## Project Structure

This is a generator for setting up Minecraft servers with the following features:
- **Custom Names**: Easily set unique names for your servers.
- **Ports**: Configure specific ports for server connections.
- **Domains**: Option to link custom domains for easy access.
- **Seeds**: Allow users to input custom world seeds for unique generations.
- **Modes**: Choose between Creative and Survival modes for gameplay options.
- **Editions**: Support for both Java and Bedrock editions of Minecraft.
- **Geyser Plugin Support**: Integration for Bedrock players to join Java servers seamlessly.

## Directory Structure
- `src/`: Source code for the server generator.
- `config/`: Configuration files for server settings.
- `scripts/`: Utility scripts for deployment.
- `tests/`: Test cases for quality assurance.
- `README.md`: Project README file explaining the setup and usage.
